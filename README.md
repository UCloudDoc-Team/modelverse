# 概览

模型服务平台 UModelVerse 旨在为客户提供快速搭建 AGI 应用的能力。我们强大的 API 让您无需关注底层算力调度、模型部署等复杂工作。仅需一个 API Key，即可轻松接入 OpenAI、Gemini 兼容的 API 接口，快速构建您的专属 AGI 应用。
